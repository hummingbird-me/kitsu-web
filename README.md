## Getting Started

### Installing Dependencies

* `yarn`

### Running Ember App

* `yarn start` (default port 4200)
* `yarn start --port 4201`

## Running Tests

* `yarn test` (tap reporter)
* `yarn test:ci` (dot reporter)
